Free Fire 국가별 점유율 시각화 — 프로토타입 3종

기존 Bump Chart가 "너무 일직선"이라 점유율 변화 크기가 안 보이는 문제 개선. 3가지 대안 제시.
① Ribbon Chart — 굵기가 점유율을 나타내는 변형 Bump Chart
순위는 그대로 유지하되, 선 굵기가 해당 월의 점유율(%)에 비례. 인도네시아가 24.2% 회복하는 순간 선이 급격히 두꺼워지는 것이 보임.
✅ 순위 + 점유율 크기를 한눈에. V자 회복이 시각적으로 드러남.
❌ 굵기가 심하게 달라지면 시각적 혼잡. 인도네시아가 너무 두꺼워질 수 있음.
② Streamgraph — 스택 영역 차트
각 국가가 "강(river)"처럼 흐르며, 폭이 점유율. 총합이 100%를 이루는 "시장 구성"을 직관적으로 보여줌.
✅ 시장 전체 구성 + 각 국가 변화를 동시에. 인도네시아 영역이 우상단에서 급격히 팽창하는 게 명확.
❌ 국가별 "순위"가 안 보임. 하위 국가는 얇아서 잘 안 보임.
③ Heatmap — 월×국가 색 밀도로 점유율 표현
가로축 월, 세로축 국가. 색 진하기가 점유율. 각 셀에 실제 숫자 표시. 인도네시아 행이 중간에 밝아졌다가 끝에서 다시 진해지는 V자 패턴이 보임.
✅ 정확한 숫자 + 패턴 동시 파악. 분석적 관점에서 가장 정보량 많음.
❌ "변화의 흐름"보다는 "특정 시점의 스냅샷"에 가까움. 덜 dynamic.
④ Enhanced Bump — 각 지점에 점유율 원(Circle) 추가
기존 Bump Chart 유지 + 각 월 교차점에 점유율 비례 원 표시. 최소 수정으로 최대 효과.
✅ 기존 디자인 연속성 유지. 원의 크기로 점유율 파악. 급격한 변화가 드러남.
❌ 원이 겹치면 혼잡. 여전히 "일직선" 느낌이 완전히 해소되진 않음.
